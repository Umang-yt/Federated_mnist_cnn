 # Federated Learning for Image Classification using CNN

This project implements Federated Learning using CNN and FedAvg on the MNIST dataset.

## Features
- Federated Learning
- CNN Model
- MNIST Dataset
- FedAvg Algorithm
- Privacy-Preserving Training

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- Scikit-Learn

## Output
- Accuracy Score
- Loss Score
- Accuracy/Loss Graph
- Confusion Matrix
